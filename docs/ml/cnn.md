# 第 16 章 卷积神经网络与视觉任务

系统讲 CNN、卷积、池化、感受野、参数共享、经典结构、目标检测、IoU、U-Net 和视觉数据集。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- CNN 适合图像，因为图像具有局部相关性和平移结构。
- 卷积核提取局部特征，靠近输入的层常学习边缘和纹理。
- 感受野是输出特征图一个位置对应输入图像的区域。
- ResNet 使用跳跃连接缓解深层网络训练困难。
- U-Net 用跳跃连接融合浅层空间信息和深层语义信息。
