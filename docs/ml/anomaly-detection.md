# 第 10 章 概率模型与异常检测

把高斯分布、多元高斯、密度估计、异常检测和高维高斯直觉放在一起。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 异常检测常通过估计 p(x) 找低概率样本。
- 一维高斯由均值和方差决定。
- 多元高斯用协方差矩阵描述特征相关性。
- 异常样本极少且类型未知时，异常检测比监督学习更合适。
- 高维高斯的大部分概率质量集中在薄球壳上，而不是均值点。

## 关键公式

$p(x)=\prod_{j=1}^{n}p(x_j;\mu_j,\sigma_j^2)$

## 考点覆盖

- 异常检测
- 高斯分布
- 多元高斯
- 高维高斯薄球壳
- 密度估计
