# 第 24 章 AI Agent 与应用系统

整理 Agent、闭环执行、RAG、工具调用和 OpenClaw 四角色。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- Agent 与传统 AI 助手的核心区别是闭环执行能力。
- Agent 通常包含规划、工具执行、记忆和反馈修正。
- RAG 通过检索外部知识减少幻觉。
- 工具调用要控制权限、记录日志并校验结果。
- OpenClaw 包含 GateKeeper、Planner、Worker、Memory。

## 考点覆盖

- AI Agent
- 闭环执行
- RAG
- OpenClaw 四角色
- 产业落地场景

## 和其他章节的关系
