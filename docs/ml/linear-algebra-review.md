# 第 2 章 数学基础

只保留机器学习真正常用的数学工具：矩阵维度、范数、梯度、链式法则、概率和贝叶斯公式。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 向量表示单个样本，矩阵表示一批样本，张量表示更高维数据。
- 矩阵乘法要始终检查维度，例如 X in R^{m x n}。
- L1 范数对应参数绝对值和，L2 范数对应参数平方和。
- 反向传播的核心是链式法则。
- 贝叶斯公式连接先验、似然和后验。

## 关键公式

$P(A \mid B)=\frac{P(B \mid A)P(A)}{P(B)}$

$\nabla_{\theta}J(\theta)$
