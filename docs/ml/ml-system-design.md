# 第 20 章 概率图模型、贝叶斯方法与因果推断

补充贝叶斯网络、马尔可夫随机场、贝叶斯线性回归、贝叶斯优化、高斯过程、因果推断和 NAS。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 贝叶斯线性回归给参数加入先验，相当于在损失里加入正则化偏好。
- 贝叶斯网络是有向图，马尔可夫随机场是无向图。
- 高斯过程可作为贝叶斯优化的代理模型。
- 核函数定义函数的平滑性和相关性假设。
- 相关性不等于因果性，随机对照试验是因果识别金标准。
- ENAS 通过权重共享提高架构搜索效率。

- 贝叶斯优化

- NAS/ENAS
