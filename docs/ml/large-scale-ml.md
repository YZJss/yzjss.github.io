# 第 25 章 机器学习系统设计与工程实践

集中整理 Precision、Recall、F1、误差分析、数据增强、分布偏移、大规模训练、在线学习和上线评估。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 准确率在类别不平衡时可能误导。
- Precision 衡量预测为正的样本中有多少是真的正。
- Recall 衡量真实正样本中有多少被找出来。
- F1 综合 Precision 和 Recall。
- 工程系统要关注数据质量、延迟、成本、监控和分布偏移。

## 关键公式

$Precision=\frac{TP}{TP+FP}$

$Recall=\frac{TP}{TP+FN}$

$F1=\frac{2PR}{P+R}$
