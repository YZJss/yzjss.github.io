# 第 1 章 机器学习概述

建立整套机器学习知识地图：任务、数据、模型、性能指标，以及监督学习、无监督学习、判别式模型和生成式模型的区别。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 机器学习定义：从经验中提升任务性能。
- 监督学习处理带标签样本，核心任务是回归和分类。
- 无监督学习不依赖标签，常见任务是聚类、降维、密度估计和异常检测。
- 判别式模型学习 P(y | x)，生成式模型学习 P(x) 或 P(x, y)。

## 考点覆盖

- 监督学习 / 无监督学习
- 回归 / 分类
- 生成式模型 / 判别式模型
- 密度估计属于无监督学习

## 和其他章节的关系

第 1 章是总览，后续章节会把这里的任务类型逐一展开。
