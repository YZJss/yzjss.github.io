# 第 6 章 模型评估与泛化

集中整理训练集、验证集、测试集、欠拟合、过拟合、偏差方差、学习曲线、超参数和数据泄漏。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 训练集用于拟合参数，验证集用于选择模型和超参数，测试集用于最终评估泛化能力。
- 训练误差和验证误差都高，通常是欠拟合。
- 训练误差低、验证误差高，通常是过拟合。
- 超参数是训练前设置的，如学习率、网络层数、正则化强度。
- 数据泄漏会让评估结果虚高。

## 考点覆盖

- 训练/验证/测试集
- 欠拟合和过拟合
- 偏差方差
- 超参数
- 数据泄漏
