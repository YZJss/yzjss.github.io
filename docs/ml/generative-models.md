# 第 21 章 生成模型

系统讲生成式模型、潜变量、自编码器、VAE、KL 散度、重参数化、GAN 和扩散模型。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 生成模型学习数据如何产生。
- VAE 使用潜变量分布，训练目标包含重构项和 KL 散度。
- 重参数化技巧让采样过程可反向传播。
- GAN 包含生成器和判别器，可能出现模式崩溃。
- 扩散模型正向加噪，反向去噪，训练时常预测添加的噪声。

## 考点覆盖

- VAE
- KL 散度
- GAN
- 扩散模型预测噪声
- 反向过程是马尔可夫链
