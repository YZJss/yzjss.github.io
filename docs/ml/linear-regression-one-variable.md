# 第 3 章 优化基础

把损失函数、梯度下降、学习率、批量/随机/小批量训练和动量单独讲清楚，为后面所有模型训练铺路。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 训练就是最小化损失函数。
- 梯度下降沿负梯度方向更新参数。
- 学习率控制每次参数更新的步长，太大可能震荡，太小收敛慢。
- 小批量梯度下降是现代深度学习最常用的训练方式。
- 动量通过累积历史梯度方向加速收敛并抑制震荡。

## 关键公式

$\theta_j \leftarrow \theta_j - \alpha \frac{\partial J(\theta)}{\partial \theta_j}$

## 考点覆盖

- 学习率作用
- 损失震荡可能是学习率太大
- Momentum 的作用
- Batch / SGD / Mini-batch 区别

## 和其他章节的关系

第 3 章讲优化，后续模型训练都会反复使用梯度下降思想。
