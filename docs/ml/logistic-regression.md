# 第 5 章 逻辑回归与分类模型

讲清二分类概率模型：sigmoid、决策边界、交叉熵、多分类 one-vs-all 和 softmax 的关系。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 逻辑回归输出 P(y=1 | x)。
- sigmoid 将线性输出压到 0 到 1。
- 决策边界由 theta^T x = 0 决定。
- 逻辑回归使用交叉熵损失。
- 多分类可以用 one-vs-all，也可以用 softmax。

## 关键公式

$h_{\theta}(x)=\frac{1}{1+e^{-\theta^{T}x}}$

$\text{softmax}(z)_k=\frac{e^{z_k}}{\sum_j e^{z_j}}$
