# 第 22 章 迁移学习与预训练模型

讲迁移学习、特征提取、微调、ImageNet 预训练、词嵌入、BERT、Embedding/Reranker/生成模型区别。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 迁移学习使用大数据预训练模型帮助小数据任务。
- 特征提取通常冻结预训练模型，只训练新分类器。
- 微调会在新任务上继续训练部分或全部参数。
- 词嵌入把词映射到低维稠密向量空间。
- BERT 预训练任务包括 MLM 和 NSP。
- Embedding 用于召回，Reranker 用于重排序，生成模型用于生成答案。

## 考点覆盖

- 迁移学习
- ImageNet 预训练 VGG + SVM
- Word Embedding
- BERT MLM/NSP
- Embedding/Reranker
