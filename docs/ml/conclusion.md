# 第 19 章 总结

这一章是课程总结，回顾前面学过的主要机器学习方法和实践建议。

## 19.1 总结和致谢

课程从监督学习开始，介绍了线性回归、逻辑回归、神经网络和支持向量机。

监督学习部分的核心问题是：

```text
给定带标签训练集，学习输入到输出的映射
```

课程还介绍了无监督学习，包括聚类和降维。

无监督学习的核心问题是：

```text
没有标签时，让算法自动发现数据结构
```

后续章节还覆盖了异常检测、推荐系统、大规模机器学习以及 Photo OCR 应用实例。

除了算法本身，课程反复强调实践方法：

- 用训练集、交叉验证集和测试集评估模型。
- 通过偏差和方差诊断问题。
- 用学习曲线判断是否需要更多数据。
- 用误差分析决定下一步。
- 用上限分析优化复杂系统。

这些方法的共同目标是：不要只凭感觉改模型，而是用实验和数据决定下一步。

## 19.2 后续学习路线

吴恩达这门课适合作为机器学习入门主线。继续往后学，可以按下面路线扩展：

| 方向 | 内容 |
| --- | --- |
| 数学基础 | 线性代数、概率、优化 |
| 深度学习基础 | 神经网络、反向传播、正则化、优化器 |
| 计算机视觉 | CNN、目标检测、图像分割 |
| 序列模型 | RNN、LSTM、Transformer |
| 图学习 | 图神经网络、消息传递、过平滑 |
| 生成模型 | VAE、GAN、扩散模型 |
| 工程实践 | PyTorch、数据处理、训练监控、部署 |

学习时可以保留一条主线：先理解任务，再理解模型，最后理解训练和评估。不要只背模型名字，要能说清楚它解决什么问题、输入输出是什么、损失函数怎么定义、如何判断效果好坏。
