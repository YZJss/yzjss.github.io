# 第 26 章 综合复习与考点索引

把全套笔记和考试题、参考书知识点建立索引，作为最后复习入口。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 第 1-12 章覆盖传统机器学习基础。
- 第 13-19 章覆盖神经网络、CNN、RNN、Transformer 和 GNN。
- 第 20-24 章覆盖贝叶斯、生成模型、预训练、大模型和 Agent。
- 第 25 章覆盖工程实践和评价指标。
- 复习时优先看易混概念：参数/超参数、PCA/特征选择、BatchNorm/LayerNorm、Embedding/Reranker。

## 考点覆盖
