## 26.4 易混概念

| 概念 | 区别 |
| --- | --- |
| 参数和超参数 | 参数由训练学习，超参数训练前设置 |
| 训练集、验证集、测试集 | 训练集拟合参数，验证集调模型，测试集最终评估 |
| 欠拟合和过拟合 | 欠拟合训练/验证都差，过拟合训练好验证差 |
| L1 和 L2 | L1 更容易稀疏，L2 更常用于权重衰减 |
| PCA 和特征选择 | PCA 生成新特征，特征选择保留原特征 |
| 反向传播和优化器 | 反向传播计算梯度，优化器根据梯度更新参数 |
| BatchNorm 和 LayerNorm | BatchNorm 依赖 batch，LayerNorm 不依赖 batch |
| BERT 和 GPT | BERT 偏理解，GPT 偏生成 |
| Embedding 和 Reranker | Embedding 负责召回，Reranker 负责重排 |
| Agent 和普通助手 | Agent 强调闭环执行和工具调用 |
