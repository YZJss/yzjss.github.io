# 第 9 章 无监督学习与聚类

讲无监督学习任务和 K-means：算法步骤、优化目标、随机初始化、选择 K 和局限。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- K-means 反复执行样本分配和簇中心更新。
- 目标是最小化样本到所属簇中心的平方距离。
- K-means 对初始化敏感，常多次随机初始化。
- 选择 K 可以用肘部法则，但也要结合业务含义。
- 聚类属于无监督学习。
