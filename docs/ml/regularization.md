# 第 7 章 正则化与模型复杂度

整理 L1、L2、权重衰减、Dropout、早停、数据增强、标签平滑，以及贝叶斯先验视角。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 正则化通过限制模型复杂度缓解过拟合。
- L2 正则化会在梯度中增加 lambda/m * theta_j。
- 通常不惩罚偏置 theta_0。
- L1 更容易得到稀疏参数，可用于特征选择。
- Dropout、早停、数据增强也属于广义正则化思想。

## 关键公式

$J(\theta)=J_0(\theta)+\frac{\lambda}{2m}\sum_{j=1}^{n}\theta_j^2$

$\frac{\partial}{\partial \theta_j}\frac{\lambda}{2m}\theta_j^2=\frac{\lambda}{m}\theta_j$
