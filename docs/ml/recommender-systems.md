# 第 12 章 推荐系统与表示学习

整理基于内容推荐、协同过滤、矩阵分解、均值归一化、冷启动，并引出 embedding 思想。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 基于内容推荐依赖物品特征和用户偏好。
- 协同过滤从用户行为中学习用户向量和物品向量。
- 评分可近似为用户向量和物品向量的内积。
- 矩阵分解是一种表示学习。
- 冷启动可用内容特征、热门推荐或主动收集偏好缓解。

## 考点覆盖

- 推荐系统
- 协同过滤
- 矩阵分解
- Embedding 思想
- 冷启动
