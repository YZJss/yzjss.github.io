# 第 15 章 深度学习训练技巧

集中讲现代深度学习训练技巧：激活函数、梯度消失爆炸、初始化、BatchNorm、LayerNorm、Dropout、优化器和学习率策略。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- ReLU 是深度学习中最常用的激活函数之一。
- BatchNorm 主要稳定训练、加速收敛。
- LayerNorm 不依赖 batch size，常用于 RNN 和 Transformer。
- 梯度消失和爆炸在深层网络中更常见。
- 学习率调度包括阶梯下降、指数衰减、余弦退火和预热。

## 考点覆盖

- ReLU
- BatchNorm
- LayerNorm
- 梯度消失/爆炸
- 学习率调度
- Adam/Momentum
