# 第 4 章 线性回归

系统讲线性回归：模型表示、平方误差、梯度下降、特征缩放、多项式特征，以及它和神经网络线性输出的关系。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 单变量线性回归是 h_theta(x)=theta_0+theta_1 x。
- 多变量线性回归可写成 h_theta(x)=theta^T x。
- 平方误差适合连续值预测。
- 特征缩放能加速梯度下降。
- 多项式特征能增强表达能力，但可能导致过拟合。

## 关键公式

$h_{\theta}(x)=\theta^{T}x$

$J(\theta)=\frac{1}{2m}\sum_{i=1}^{m}(h_{\theta}(x^{(i)})-y^{(i)})^2$
