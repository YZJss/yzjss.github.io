# 第 11 章 支持向量机与核方法

讲 SVM 的大间隔思想、支持向量、核函数、RBF 核，以及它和逻辑回归的区别。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- SVM 关注最大化分类间隔。
- 支持向量是离边界最近、决定边界的样本。
- 核函数允许模型隐式进入高维特征空间。
- C 控制训练错误和间隔大小的权衡。
- SVM 更强调边界，逻辑回归更强调概率。

## 考点覆盖

- SVM
- 大间隔
- 核函数
- RBF 核
