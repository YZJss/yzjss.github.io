# 第 8 章 特征工程、特征选择与降维

把特征缩放、特征构造、卡方检验、RFE、L1 特征选择和 PCA 放到一个系统章节。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 特征工程把原始数据转换成更适合学习的表示。
- 特征选择保留原始特征子集。
- PCA 生成原特征的线性组合，不是选择原特征。
- PCA 是无监督线性降维方法，寻找方差最大的投影方向。
- 降维可用于压缩、可视化和去噪，但可能丢失有用信息。

## 考点覆盖

- PCA
- 卡方检验
- 递归特征消除
- L1 正则化特征选择
- PCA 与特征选择区别
