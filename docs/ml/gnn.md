# 第 19 章 图神经网络

讲图数据、邻接矩阵、消息传递、邻居聚合、GCN/GAT/GraphSAGE、过平滑和过挤压。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- GNN 用于非欧几里得结构数据。
- 核心机制是消息传递和邻居聚合。
- 聚合函数通常需要对节点顺序不敏感。
- 多层消息传递后节点表示可能过于相似，称为过平滑。
- GNN 可用于节点分类、边预测和图分类，不只用于节点分类。

## 考点覆盖

- GNN
- 消息传递
- 邻居聚合
- 过平滑
- 非欧几里得数据
- 节点顺序不变性
