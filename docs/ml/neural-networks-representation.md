# 第 13 章 神经网络表示

只讲神经网络如何表示函数：神经元、层、参数矩阵、前向传播、隐藏层特征和多分类输出。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- 神经元可以看作线性组合加非线性激活。
- 激活函数负责引入非线性。
- Theta^(l) 的维度是 s_{l+1} x (s_l + 1)。
- 前向传播逐层计算 z 和 a。
- 隐藏层可以自动学习中间特征。

## 考点覆盖

- 神经元
- 偏置作用
- 激活函数引入非线性
- 参数矩阵维度
- 隐藏层特征
