# 第 17 章 序列模型

整理 RNN、BPTT、梯度消失爆炸、LSTM、GRU、遗忘门和 Teacher Forcing。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- RNN 使用隐藏状态保存历史信息。
- BPTT 是沿时间展开后的反向传播，复杂度与时间步长度有关。
- LSTM 用门控机制缓解长序列训练困难。
- 遗忘门决定从细胞状态中丢弃哪些信息。
- Transformer 出现的动机之一是 RNN 难并行且长距离依赖困难。

## 考点覆盖

- RNN
- BPTT
- LSTM 遗忘门
- GRU
- 序列任务
