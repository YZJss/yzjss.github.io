# 第 23 章 大模型基础

整理 GPT/BERT 区别、温度参数、MoE、SFT、RL 后训练、推理能力和幻觉。

## 学习主线

本章在整套笔记中的作用是把相关知识点放到同一个框架里，先理解它解决什么问题，再理解模型或方法如何工作，最后记住考试和工程中容易混淆的点。

## 核心内容

- BERT 是 Encoder-only，更适合理解；GPT 是 Decoder-only，更适合生成。
- 温度越高，输出越随机，创造性更强。
- MoE 在相同推理成本下可拥有更大总参数量。
- SFT 用人工示例微调，RL 可基于奖励信号进一步后训练。
- 幻觉需要通过检索、工具、约束和审核缓解。
