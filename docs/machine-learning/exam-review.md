# 考试题整理

## 单选题高频考点

- 激活函数负责引入非线性。
- Batch Normalization 可以加速网络收敛，稳定训练过程。
- Transformer 的自注意力用于捕捉序列依赖。
- 学习率控制每次参数更新的步长。
- ResNet 通过跳跃连接缓解深层网络训练困难。
- F1 分数综合精确率和召回率。

## 多选题高频考点

- 防止过拟合：增加数据、L2 正则化、Dropout、数据增强。
- 无监督学习：聚类、降维、密度估计。
- CNN 常见操作：卷积、池化、全连接。
- 贝叶斯核心概念：先验分布、似然函数、后验分布。

## 填空题高频考点

- 测试集用于最终评估泛化能力。
- 反向传播计算损失函数关于权重的梯度。
- IoU 衡量预测边界框与真实边界框之间的重合程度。
- BERT 预训练任务包括掩码语言模型和下一句预测。
