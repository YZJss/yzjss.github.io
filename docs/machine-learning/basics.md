# 机器学习基础

## 一句话理解

机器学习是让模型从数据中学习规律，并在未见过的数据上做出预测或决策。

## 核心概念

- 训练集：用于训练模型参数。
- 验证集：用于调参和选择模型。
- 测试集：用于最终评估泛化能力。
- 泛化能力：模型在新数据上的表现。
- 过拟合：训练集表现好，验证集或测试集表现差。
- 欠拟合：训练集和验证集表现都不好。

## 常考点

- 测试集用于最终评估模型的泛化能力。
- 训练误差和验证误差都很高，通常说明模型欠拟合。
- 训练误差低而验证误差高，通常说明模型过拟合。

## 易错点

- 验证集不是最终评估集。
- 模型在训练集准确率高，不代表泛化能力强。
