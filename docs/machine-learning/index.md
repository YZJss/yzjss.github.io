# 机器学习笔记

这套笔记按章节整理，目标是同时服务三件事：

- 建立机器学习和深度学习的知识体系。
- 根据题库反推常考点和易错点。
- 用 PyTorch 示例把关键概念落到代码里。

## 建议学习顺序

1. 机器学习基础
2. 监督学习
3. 神经网络基础
4. 优化与正则化
5. CNN、RNN、Transformer
6. 贝叶斯方法、生成模型、GNN
7. 大模型与 Agent
8. PyTorch 实战和考试题整理
