# 神经网络基础

## 一句话理解

神经网络通过多层线性变换和非线性激活函数学习复杂函数。

## 核心组件

- 权重：控制输入特征的重要性。
- 偏置：允许决策边界平移，提高模型灵活性。
- 激活函数：引入非线性。
- 损失函数：衡量预测和真实标签之间的差距。

## 常见激活函数

| 激活函数 | 特点 |
| --- | --- |
| Sigmoid | 输出范围为 `(0, 1)` |
| Tanh | 输出范围为 `(-1, 1)` |
| ReLU | 深度学习中常用，计算简单 |
| Leaky ReLU | 缓解 ReLU 死亡问题 |

## 常考点

- 激活函数负责引入非线性。
- 偏置参数的主要作用是允许决策边界平移。
