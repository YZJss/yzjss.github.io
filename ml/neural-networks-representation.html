第 12 章 神经网络基础 ​

神经网络是由许多神经元连接起来的模型。它不是只做一次线性计算，而是把“线性组合 + 非线性变换”重复多次，让模型能够表示更复杂的函数。

本章只讲最基础、最常用的一类神经网络：全连接神经网络。后面的卷积神经网络、循环神经网络、Transformer 都可以看成在这个基础上针对特定数据结构做出的改造。

12.1 神经元 ​

一个神经元接收输入 x1,x2,…,xn，给每个输入分配一个权重，然后再加上偏置。

线性部分为：

z=w1x1+w2x2+⋯+wnxn+b

也可以简写为：

z=wTx+b

其中：

符号含义
x输入特征
w权重
b偏置
z加权求和结果

偏置 b 的作用是让函数可以平移。没有偏置时，模型会受到更强限制，很多边界或函数形状无法自然表示。

神经元在线性部分之后还会接一个激活函数：

a=g(z)

a 是神经元的输出，也叫激活值。

12.2 激活函数 ​

如果神经网络只有线性计算，那么多层网络仍然等价于一层线性模型。原因是多个线性变换叠加以后，结果还是线性变换。

例如：

W2(W1x)=(W2W1)x

所以神经网络必须加入非线性激活函数。激活函数决定了神经元如何把线性结果 z 变成输出 a。

常见激活函数：

激活函数公式特点
sigmoidσ(z)=11+e−z输出在 0 到 1，常用于二分类输出层
tanhtanh⁡(z)=ez−e−zez+e−z输出在 -1 到 1，零中心
ReLUmax(0,z)计算简单，隐藏层最常用

sigmoid 和 tanh 在 z 很大或很小时会进入饱和区，导数接近 0，深层网络中容易造成梯度消失。ReLU 在正半轴导数为 1，训练深层网络时更常用。

ReLU 也有问题：如果某个神经元长期落在负半轴，它的输出和梯度都可能为 0，这个神经元可能不再更新。实际中可以用较小学习率，或使用 Leaky ReLU、ELU 等变体缓解。

12.3 全连接神经网络 ​

全连接神经网络又叫多层感知机。它的特点是：相邻两层之间的每个神经元都相互连接。

一个简单的全连接网络可以写成：

text
输入层 -> 隐藏层 -> 输出层

输入层只负责接收特征，不做真正的学习。隐藏层负责把原始特征转换成新的表示。输出层负责给出最终预测。

假设输入是房屋面积、房间数、楼层等特征，隐藏层中的不同神经元可能学习到不同组合：

  • 有的神经元更关注面积。
  • 有的神经元更关注位置和楼层。
  • 有的神经元更关注多个特征共同出现时的影响。

这些神经元的输出再传给下一层，下一层继续组合它们。神经网络的表达能力就来自这种逐层组合。

全连接网络适合处理已经整理成特征向量的数据。对于图像、文本、语音这类有明显结构的数据，直接使用全连接网络通常不是最好的选择，所以后面会引入 CNN、RNN、Transformer。

12.4 神经网络参数初始化 ​

神经网络的权重不能全部初始化为 0。

如果同一层所有神经元的权重完全相同，它们在前向传播中会得到相同输出，在反向传播中也会得到相同梯度。训练之后这些神经元仍然相同，相当于多个神经元只学到同一个东西。

这叫对称性问题。解决办法是随机初始化权重。

初始化的目标不是让模型一开始就预测准确，而是让前向传播的信号和反向传播的梯度都比较稳定。常见方法：

方法适合场景思想
Xavier 初始化sigmoid、tanh让每层输入和输出的方差尽量稳定
He 初始化ReLU考虑 ReLU 会截断一部分负值

偏置通常可以初始化为 0，权重需要用小随机数初始化。

12.5 前向传播 ​

前向传播就是从输入层开始，一层一层算到输出层。

对某一层来说，计算过程很简单：

  1. 先做线性组合：z=wTx+b。
  2. 再过激活函数：a=g(z)。

多层网络就是重复这个过程：

text
输入 x
-> 第 1 层线性组合
-> 第 1 层激活
-> 第 2 层线性组合
-> 第 2 层激活
-> 输出预测

二分类任务中，输出层常用 sigmoid：

y^=σ(z)

y^ 可以理解为样本属于正类的概率。

多分类任务中，输出层常用 softmax：

y^k=ezk∑j=1Kezj

其中 K 是类别数，y^k 表示第 k 类的预测概率。

12.6 反向传播 ​

训练神经网络时，前向传播得到预测值，损失函数衡量预测值和真实标签的差距。反向传播要做的事是：计算每个参数应该往哪个方向调整，才能让损失下降。

反向传播的核心是链式法则。

如果损失是 L，某个参数是 w，那么训练需要计算：

∂L∂w

这个值表示：参数 w 变化一点，损失 L 会如何变化。

可以把反向传播理解为两步：

  1. 先计算输出层的误差。
  2. 再把误差一层层传回前面的层。

每一层的权重梯度都可以直观理解成：

text
当前层误差 × 上一层输出

所以某个连接权重的更新，既取决于当前层错得多不多，也取决于上一层给了多大的输入。

现代深度学习框架会自动完成反向传播，但理解它仍然重要。因为损失不下降、梯度消失、梯度爆炸、学习率不合适等问题，都要从前向传播和反向传播的角度排查。

12.7 完整训练过程 ​

神经网络的完整训练过程如下：

  1. 准备训练集、验证集和测试集。
  2. 选择网络结构，例如隐藏层层数、每层神经元个数、激活函数。
  3. 随机初始化权重，初始化偏置。
  4. 前向传播，得到预测值。
  5. 计算损失函数。
  6. 反向传播，计算参数梯度。
  7. 用优化算法更新参数。
  8. 在验证集上观察效果，调整模型结构、学习率、正则化等超参数。
  9. 最后在测试集上评估模型泛化能力。

一次前向传播和一次反向传播只完成一次参数更新。实际训练时，这个过程会在许多 mini-batch 上重复很多轮。

完整流程可以概括为：

text
初始化参数 -> 前向传播 -> 计算损失 -> 反向传播 -> 更新参数 -> 重复训练

神经网络真正难的地方不在于单个公式，而在于训练过程是否稳定：初始化是否合适，激活函数是否合理，学习率是否过大或过小，训练集和验证集表现是否一致。这些问题会在后续深度学习章节继续展开。
